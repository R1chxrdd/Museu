# Museu

Projeto web desenvolvido com Flask com temática de museu, criado como atividade acadêmica para praticar desenvolvimento web com Python.

## Sobre o Projeto

O sistema foi desenvolvido com o objetivo de apresentar páginas informativas e estruturação de um site temático utilizando Flask, HTML e CSS, aplicando conceitos de rotas, templates e organização de projeto web.

## Funcionalidades

- Navegação entre páginas
- Estrutura temática de museu
- Interface web organizada
- Utilização de templates
- Separação entre front-end e back-end
- Rotas com Flask

## Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3
- Jinja2


## Conceitos Aplicados

- Desenvolvimento web com Flask
- Rotas e renderização de páginas
- Templates dinâmicos
- Organização de arquivos estáticos
- Estrutura MVC/MVT adaptada

## Objetivo Acadêmico

Projeto desenvolvido para fins educacionais, com foco em aprendizado prático de aplicações web em Python.

```bash
# Clonar repositório

# Entrar na pasta do projeto
